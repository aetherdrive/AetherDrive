# AetherDrive - Motoren v3

**AetherDrive** er en avansert motor for moderne applikasjoner, laget med Node.js.  
Denne versjonen (v3) gir en enkel startstruktur, lett å utvide og deploye.

## Komme i gang

1. Klon repoet:
```bash
git clone <repo-url>
cd AetherDrive
```

2. Installer dependencies:
```bash
npm install
```

3. Kjør motoren:
```bash
npm start
```

## Struktur
- `src/` - koden til motoren
- `tests/` - testskript
- `package.json` - Node.js prosjektkonfig
